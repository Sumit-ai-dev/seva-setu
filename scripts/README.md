# Scripts

Utility and data migration scripts for the Nexus Health platform.

| Script | Purpose |
|---|---|
| `replace_demo_locations.py` | Migrates demo patient data from Maharashtra to Karnataka coordinates |
| `replace_labels.py` | Updates UI label strings across the frontend |
| `replace_locale.py` | Updates locale/language strings for Kannada support |
| `replace_triage_labels.py` | Updates triage severity label strings |

## Usage

```bash
cd scripts/
python replace_demo_locations.py
```
